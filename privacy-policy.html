# Política de Privacidad **Última actualización: [Fecha]** Esta aplicación usa la API de Google Calendar para permitir a los usuarios gestionar eventos de forma automatizada en n8n. No almacenamos ni compartimos ningún dato personal. Toda la información del usuario permanece dentro de Google Calendar y no se guarda en servidores externos. ## 📌 Datos recopilados - Fechas y horarios de eventos (para crear eventos en Google Calendar). - Títulos y descripciones de eventos (si el usuario los proporciona). ## 🔒 Seguridad - La aplicación usa OAuth 2.0 para autenticación segura con Google. - No compartimos ni vendemos datos de usuario.
